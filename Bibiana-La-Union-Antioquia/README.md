# Propuesta Integral - Hotel Boutique

Página web tipo presentación para GitHub Pages.

## Cómo subirla a GitHub

1. Crea un repositorio nuevo o abre el repositorio donde quieras publicarla.
2. Sube `index.html` y la carpeta `assets` completa.
3. En GitHub, entra a **Settings > Pages**.
4. En **Build and deployment**, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guarda los cambios.
6. GitHub te entregará una URL pública en unos minutos.

## Archivos incluidos

- `index.html`: página principal.
- `assets/slide-01.webp` a `slide-04.webp`: imágenes optimizadas.
- `assets/slide-01.jpg` a `slide-04.jpg`: respaldo para navegadores.

## Edición rápida

Puedes abrir `index.html` en cualquier editor y cambiar textos, botones o títulos.
